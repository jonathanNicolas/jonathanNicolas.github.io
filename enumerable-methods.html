Enumerable#map

Describing and explaining map

Map allows to iterate over every element and whatever we return in that block gets added(mapped) to a new array

Example

nums = Array[1, 2, 3, 4,5]

nums.map{|num| num.to_s}

=> ["1", "2", "3", "4","5"]

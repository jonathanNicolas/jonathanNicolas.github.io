Program Structure

The else keyword can be used, together with if, to create two seperate alternatives

        var theNumber = Number(prompt("Pick a number", ""));
        if (!isNan(theNumber))
          alert("Your number is the square root of " + theNumber * theNumber)
        else
          alert("Hey. why didn't you give me a number?");

Looping control flow allows us to go back to some point in the program where we were before and repeat it with our current program state

  •  While loop
        var result = 1;
        var counter = 0;
        while (counter < 10){
          result = result * 2;
          counter = counter + 1;
        }
        
  •  For loop
        for (var current = 20; ; current ++) {
          if (current % 7 == 0)
            break;
        }

Data Structures: Objects and Arrays

Data Sets

Javascript provides a data type specifically for storing sequences of values it is called an array

Properties

The two most common way to access properties in Javascript are with the dot notation and with square brackets. Both value.x and value[x] access property on value

Objects

Values of the type object are arbitrary collections of properties, one way to create an object is by using a brace notation

          var day1 = {
          squirrel: false,
          events: ["work", "touched tree", "pizza"]
          }

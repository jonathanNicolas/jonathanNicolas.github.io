Arrays & Hashes

Describing and comparing arrays and hashes

Ruby arrays are ordered, integer-indexed collections of any object. Each element in an array is associated with and referred to by an index.

Example

nums = Array[1, 2, 3, 4,5]

Hashes are similar to arrays in that they are indexed collection of object references. However, while you index arrays with integers, you can index a hash with objects of any types: strings, regular expressions, and so on. When you store a value in a hash, you actually supply two objects - the index (normally called the key) and the value. You can subsequently retrieve the value by indexing the hash with the same key. The values in a hash can be objects of any type.

Example

names = Hash.new
names[1] = "Jonathan"
names[2] = "Simon"
